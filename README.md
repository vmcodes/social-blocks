# Social Blocks

![Social Blocks](logo.png)

## Repo for the Social Blocks DApp on ArcBlock

### Contributing

- Please feel free to inquire about any unassigned issues or make a pull request after forking the repository.

### Local Development

- Build backend

```shell
docker compose up -d
```

- Start frontend

```shell
yarn dev:install
yarn dev:client
```
